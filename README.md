# SafeWalk – Personal Safety App

## Project Overview

SafeWalk is a personal safety mobile application developed to provide security and assistance to users while traveling alone. The application is designed for students, women, employees, and solo travelers who may require immediate help during emergencies. SafeWalk integrates real-time location tracking, emergency communication, and journey monitoring features into a single platform. Users can share their live location with trusted contacts, send emergency alerts through an SOS button, and track their journeys until they safely reach their destinations. The application focuses on simplicity, accessibility, and quick response during critical situations.

## Objective

The main objective of SafeWalk is to improve personal security by enabling users to stay connected with trusted contacts and receive immediate assistance during emergencies. The application aims to reduce response time and provide peace of mind through location sharing and safety notifications.

## Features

- User Registration and Login
- Emergency Contacts Management
- Live GPS Location Tracking
- SOS Emergency Alerts
- Safe Walk Journey Tracking
- Safety Check-In System
- Safety History Records
- Push Notifications
- Profile and Settings Management
